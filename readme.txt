Student Registration System
1. Technology Used: NodeJS, MongoDB, HTML, CSS, MDBootstrap
2. Instructions for using:
I. npm install
II. node app.js

3. API's:-

	I. /users/login    For Students login
	II. /users/register For Students registration
	III. /myprofile	    For Students Profile
	IV.  /myprofile/update	For Updation of student Profile

4. Functionality:-
	I. Students can register and login themselves. [Passport Authentication system has been used
	II. Only after registration, Students can update their profile.
